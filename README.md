Fashion Studio 是一个面向时尚内容创作者的图片处理工具。


## 当前版本

Build001


已完成：

- 基础项目结构
- Windows桌面GUI

## 安装


安装Python 3.11


运行：

pip install -r requirements.txt



启动：

python main.py
